Resources

Books on categories and types for the TFG

Books on category theory

Categories for the working mathematician - Saunders Mac Lane

A complete course on category theory.

  • Category theory.
  • Monoidal categories.
  • 2-categories.
Introduction to Higher order categorical logic - Lambek

A course in categorical logic.

  • Cartesian closed categories.
  • Type theory and toposes.
Conceptual Mathematics: a first introduction to categories - Lawvere

A course on category theory, oriented towards categorical logic.

  • Basic cateogory theory
  • Categorical logic, toposes
Practical foundations of Mathematics - Paul Taylor

A type-oriented foundation of mathematics.

  • Type theory.
  • Cartesian closed categories.
  • Algebra of dependent types.
Sheaves in Geometry and Logic - MacLane, Moerdijk

A course on sheaves, topoi and logic. It assumes a categorical background.

  • Grothendieck topologies and sheaves.
  • Topoi and logic.
  • Classifying topoi.
Introduction to category theory and categorical logic - Thomas Streicher

Basic notions of category theory and a bit on λ-calculus, cartesian-closed categories and toposes.

  • Basic category theory
  • Cartesian closed categories and λ-calculus
  • Logic of toposes
An introduction to topos theory - Kostecki
Toposes, Triples and Theories - Barr & Wells

Books on λ-calculus

Practical foundations for Programming Languages - Robert Harper

A complete course on types, syntax and programming languages.

  • Judgements and rules.
  • Levels of syntax.
  • Data types.
  • System F.
  • Types and propositions.
  • Laziness, paralellism and concurrency.
Types and programming languages - Benjamin C. Pierce
Lecture notes on the lambda calculus - Peter Selinger
Homotopy Type Theory Book - The univalent foundations program

The foundational book on homotopy type theory.

Lambda Calculus and Combinators. An Introduction - Hindley, Seldin

Basic introduction to the λ-calculus.

  • Combinatory logic.
  • Formal theories of λ-calculus.
  • Typing.
  • Models of λ-calculus.
Programming in Martin-Löf's Type Theory - Nordström, Petersson

Martin-Löf type theory and how to write languages based on it. Based on sets.

CS410 Advanced Functional Programming - Connor McBride

A course on Agda.

  • Propositions as types.
  • Dependent types.
Software foundations - Benjamin C.

syscall with literals

Add a "literal" type.

Output
churchLiteral n = n (\k . syscall ("increment.sh" k)) "0"
printLiteral l = syscall2 ""
Input

It would use $ as an antiliteral. Any string started on $ would be interpreted as a lambda term. 

readChurch = syscall "readChurch"

readChurch would return something as "@(\s.\z.s (s (s z)))". Two literals applied one over thw other would be reinterpreted as their concatenation.
